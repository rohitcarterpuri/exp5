## 📊 Project Overview

Advanced Artificial Neural Network (ANN) implementation for customer churn prediction with cross-validation, hyperparameter optimization, and model checkpointing.

### Key Features
- ✅ **Cross Validation**: Stratified K-Fold validation (5-fold default)
- ✅ **Grid Search**: Comprehensive hyperparameter tuning
- ✅ **Checkpointing**: Automatic model saving and early stopping
- ✅ **Class Imbalance Handling**: SMOTE and oversampling techniques
- ✅ **TensorBoard Integration**: Real-time training monitoring
- ✅ **Comprehensive Metrics**: Accuracy, Precision, Recall, F1, AUC-ROC
